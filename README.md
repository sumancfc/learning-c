## Learning C Programming Language

### Memory Size in C
| Data Type |     Size     |
|:---------:|:------------:|
|    int    | 2 or 4 bytes |
|   float   |   4 bytes    |
|  double   |   8 bytes    |
|   char    |    1 byte    |

### Type of Conversion in C
- Implicit Conversion (Automatically)
- Explicit Conversion (Manually)
